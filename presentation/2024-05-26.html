编程工具¶

Python/Conda

Python 是深度学习研究的第一语言，因为它易于学习和使用，同时拥有强大的第三方库支持，可以以极高效率轻松处理数据，因此在机器学习和深度学习领域的应用非常广泛。
Conda 是一个开源的包管理和环境管理系统，它快速安装、运行和更新软件包及其依赖项，可以便捷的处理各个 Python 项目之间的依赖问题。

Read More

Jupyter Notebook

是一个交互式笔记本，支持运行多种编程解释型语言。 它的本质是一个 Web 应用程序，便于创建和共享文学化程序文档，支持实时代码，数学方程，可视化和 markdown。 用途包括：数据清理和转换，数值模拟，统计建模，机器学习等等 ，非常适合数据分析和项目演示 。

Read More

Git/Github

Git 是一种可以智能追踪文件中的更改的版本控制系统，可以有效高速地处理从各种大小的项目版本管理。
GitHub 是一个面向开源及私有软件项目的托管平台，只支持Git作为唯一的版本库格式进行托管，拥有庞大的开发者社区， 也是一个学习和分享的平台，可以在其中探寻感兴趣的项目，交流编程经验，参与开源精神。

Read More

深度学习¶

Pytorch

是很受欢迎的开源的深度学习框架，用于构建神经网络。它是基于Torch的Python版本，支持GPU加速和自动求导。 它由Facebook的人工智能研究团队开发，使用Python编写，易用、灵活、功能强大。 它常用于图像识别、自然语言处理等机器学习应用程序，具备张量操作、自动求导机制、神经网络创建、数据处理、模型训练、测试、保存和加载等内容。

Read More

TensorFlow

是Google开发的受欢迎的开源机器学习框架，它具有快速、灵活并适合产品级大规模应用等特点，让每个开发者和研究者都能方便地使用人工智能来解决多样化的挑战。 其中 tf.keras 是用于构建和训练深度学习模型的高阶 API，利用此 API，可实现快速原型设计、先进的研究和生产

Read More

scikit-learn

是基于 Python 语言的机器学习工具。它建立在 NumPy, SciPy, Pandas 和 Matplotlib 之上， 里面的 API 的设计非常好，所有对象的接口简单，很适合新手上路。 在 Sklearn 里面有六大任务模块：分别是分类、回归、聚类、降维、模型选择和预处理。

可视化网络模型¶

TensorBoard是一个可视化工具，它可以用来展示网络图、张量的指标变化、张量的分布情况等。特别是在训练网络的时候，我们可以设置不同的参数（比如：权重W、偏置B、卷积层数、全连接层数等），使用TensorBoader可以很直观的帮我们进行参数的选择。
